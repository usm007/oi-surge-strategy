How It Works

Signal Generation
At market close, fetch Nifty option chain from NSE. Compute OI change % for each strike: (Current OI − Previous OI) / Previous OI × 100. Filter strikes with OI ≥ 1,00,000. Pick the strike with highest absolute OI change %.
Execution
Next trading day, buy 1 lot (65 units) at 9:15 AM open. Sell at 3:30 PM close. Same-day only — no overnight hold.
Costs
Brokerage ₹20×2, STT 0.05%, Transaction charges 0.053%, GST 18%. Typical ₹50–₹150 per trade.
Skip Rules
Skip if entry >₹300 AND |OI%| > 300% (Extreme OI Trap). Skip if entry >₹1,500 AND OI% < -90% (OI Collapse).
Capital
₹20,000 daily limit. If entry premium exceeds ₹308, skip. No compounding — profits taken aside.
